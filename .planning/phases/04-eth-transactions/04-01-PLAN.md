       - Export `estimateGas(network, from, to, value: bigint): Promise<bigint>` -- calls `eth_estimateGas`, applies 20% buffer (`estimate + estimate / 5n`), then enforces floor (`buffered > GAS_FLOOR ? buffered : GAS_FLOOR`). Note: buffer applied BEFORE floor per research doc.
       - Export `getFeeParams(network): Promise<{ baseFee: bigint; priorityFee: bigint; maxFeePerGas: bigint }>` -- calls `eth_gasPrice` for baseFee, `eth_maxPriorityFeePerGas` for priorityFee, computes `maxFeePerGas = baseFee + priorityFee`

    5. Create barrel `src/features/wallet/rpc/index.ts` re-exporting from provider and gas.

    6. Create `src/features/wallet/tx/build.ts`:
       - Import `Transaction` from `micro-eth-signer`
       - Export `buildAndSignTransaction(params: { to: string; value: bigint; nonce: bigint; gasLimit: bigint; maxFeePerGas: bigint; maxPriorityFeePerGas: bigint; chainId: number; privateKey: Uint8Array }): string` -- uses `Transaction.prepare()` to build EIP-1559 Type 2 tx, then `tx.signBy(privateKey)`, returns hex string with `0x` prefix (check if `toHex()` includes prefix; add if missing).
       - Export `calculateMaxSend(balanceWei: bigint, gasLimit: bigint, maxFeePerGas: bigint): bigint` -- `balance - (gasLimit * maxFeePerGas)`, returns 0n if negative.
       - Export `formatEth(wei: bigint): string` -- up to 4 significant decimal places, trim trailing zeros per context decision.
       - Export `formatUsd(amount: number): string` -- Intl.NumberFormat USD formatting.
       - Export `parseEthToWei(eth: string): bigint` -- parse user input "1.5" to wei bigint. Handle decimal precision carefully (multiply string parts, don't use floating point).
       - Export `validateAddress(input: string): { valid: boolean; error?: string }` -- use `addr` from micro-eth-signer for validation (isValid check), also check 0x prefix and 42-char length.

    7. Create barrel `src/features/wallet/tx/index.ts`.

       - Export `getEthPrice(): Promise<number>` -- returns cached if < 60s old, otherwise fetches fresh. Returns 0 on fetch error (don't break the flow for a price failure).

  <name>Task 2: Extend message types + add all transaction background handlers</name>
  <files>
    src/features/wallet/types.ts,

  <action>
    1. Extend `WalletMessage` union in `src/features/wallet/types.ts` with new variants:
       ```typescript
       | { type: 'wallet:getBalance'; accountIndex: number }
       | { type: 'wallet:estimateGas'; to: string; value: string; accountIndex: number }  // value as hex string (BigInt not serializable); accountIndex for correct from address
       | { type: 'wallet:getFeeParams' }
       | { type: 'wallet:getEthPrice' }
       | { type: 'wallet:sendTransaction'; to: string; value: string; accountIndex: number }  // value as hex string
       ```

    2. Extend `WalletResponse` union with new variants:
       ```typescript
       | { type: 'wallet:balance'; balanceWei: string; balanceEth: string }  // hex string for wei
       | { type: 'wallet:gasEstimate'; gasLimit: string; maxFeePerGas: string; maxPriorityFeePerGas: string; estimatedFeeWei: string; estimatedFeeEth: string }
       | { type: 'wallet:feeParams'; baseFee: string; priorityFee: string; maxFeePerGas: string }
       | { type: 'wallet:ethPrice'; usd: number }
       | { type: 'wallet:txResult'; success: boolean; txHash: string; explorerUrl: string; error?: string }
       ```
       All bigint values serialized as hex strings with 0x prefix across the message boundary.

    3. Add `RecentAddress` type:
       ```typescript
       export interface RecentAddress {
         address: string;
         timestamp: number;
       }
       ```

    4. In `src/entrypoints/background.ts`, import the new modules:
       - `rpcCall, NETWORKS, getExplorerTxUrl` from rpc
       - `estimateGas, getFeeParams` from rpc
       - `buildAndSignTransaction, formatEth` from tx
       - `getEthPrice` from price

    5. Add handler functions:

       `handleGetBalance(accountIndex)`:
       - Get session, get network from chrome.storage.local
       - Find account by index in session.accounts
       - Call `eth_getBalance(address, 'latest')` via rpcCall
       - Return `wallet:balance` with hex wei and formatted ETH string

       `handleEstimateGas(to, valueHex, accountIndex)`:
       - Get session, get network
       - Find account by accountIndex in session.accounts for `from` address (same pattern as handleGetBalance)
       - Call `estimateGas(network, account.address, to, BigInt(valueHex))`
       - Call `getFeeParams(network)` for fee data
       - Calculate `estimatedFee = gasLimit * maxFeePerGas`
       - Return `wallet:gasEstimate` with all values as hex strings

       `handleGetFeeParams()`:
       - Get network, call `getFeeParams(network)`
       - Return `wallet:feeParams`

       `handleGetEthPrice()`:
       - Call `getEthPrice()`
       - Return `wallet:ethPrice`

       `handleSendTransaction(to, valueHex, accountIndex)`:
       - Get session, get network
       - Derive keypair for accountIndex using `deriveAccount(hexToBytes(session.seed), accountIndex)`
       - Fetch nonce: `eth_getTransactionCount(address, 'pending')` via rpcCall (TX-15)
       - Fetch fee params via `getFeeParams(network)`
       - Estimate gas via `estimateGas(network, address, to, value)`
       - Build + sign tx via `buildAndSignTransaction({ to, value, nonce, gasLimit, maxFeePerGas, maxPriorityFeePerGas, chainId, privateKey })`
       - Submit: try `realtime_sendRawTransaction` first (TX-07)
         - On success: extract txHash from receipt.transactionHash
         - On error containing "realtime" or on timeout (wrap in Promise.race with 10s timeout): fallback to `eth_sendRawTransaction` + poll `eth_getTransactionReceipt` every 500ms for up to 10s (TX-08)
       - Return `wallet:txResult` with txHash and explorerUrl (TX-09)
       - On any error: return `wallet:txResult` with success=false and error message
       - After successful send: save recipient to recent addresses in chrome.storage.local (max 10, deduplicated, sorted by most recent)

    6. Add all new cases to the `handleWalletMessage` switch statement.

    7. Add helper `saveRecentAddress(address: string)` that reads `recentAddresses` from chrome.storage.local, deduplicates, prepends new entry, trims to 10, saves back.

    8. Add helper `getNetworkPreference(): Promise<'mainnet' | 'testnet'>` that reads from chrome.storage.local and defaults to 'mainnet'.
  </action>

All types serialized as hex strings across message boundary.
