   Implementation (~80 lines):
   - Color palette: `['#01888C', '#FC7500', '#034F5D', '#F73F01', '#FC1960', '#C7144C', '#F3C100', '#1598F2', '#2465E1', '#F19E02']`
   - `addressToSeed(address: string): number` -- `parseInt(address.slice(2, 10), 16)`
   - Simple PRNG: `createPrng(seed: number)` -- linear congruential generator: `s = (s * 16807) % 2147483647; return (s - 1) / 2147483646`
   - `shiftHue(hexColor: string, degrees: number): string` -- convert hex to HSL, shift H, convert back to hex
   - Component: `Jazzicon({ address, size = 32 }: { address: string; size?: number })`
     - Seed from address, create PRNG
     - Shift hue for all palette colors by `rand() * 360`
     - Pick background color randomly from palette
     - Generate 4 rectangles with random position/rotation/color
     - Render as SVG with clipPath circle mask
     - **IMPORTANT:** Use unique clipPath ID per address to avoid conflicts when multiple jazzicons render: `clip-${addressToSeed(address)}`
