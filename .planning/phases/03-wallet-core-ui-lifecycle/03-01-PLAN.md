1. **`src/lib/password-strength.ts`** -- Password strength calculator:
   ```typescript
   export type Strength = 'weak' | 'medium' | 'strong';

   export function getPasswordStrength(password: string): Strength {
     if (password.length < 8) return 'weak';
     let score = 0;
     if (password.length >= 12) score++;
     if (password.length >= 16) score++;
     if (/[a-z]/.test(password)) score++;
     if (/[A-Z]/.test(password)) score++;
     if (/[0-9]/.test(password)) score++;
     if (/[^a-zA-Z0-9]/.test(password)) score++;
     if (score <= 2) return 'weak';
     if (score <= 4) return 'medium';
     return 'strong';
   }
   ```

2. **`src/features/wallet/store.ts`** -- Zustand store:

   **Screen type:**
   ```typescript
   export type Screen =
     | 'loading' | 'welcome'
     | 'create-password' | 'seed-phrase' | 'confirm-seed'
     | 'import-seed' | 'import-password'
     | 'main' | 'receive' | 'lock' | 'settings' | 'about';
   ```

   **Store interface:** Navigation (screenStack, currentScreen, push, pop, replace), wallet state (isLocked, accounts, activeAccountIndex, network, sidebarOpen), actions (initialize, setAccounts, setActiveAccount, setNetwork, toggleSidebar).

   **initialize():**
   1. Start on 'loading' screen
   2. Check `chrome.storage.local.get('vault')` -- if no vault, set screen to 'welcome', return
   3. If vault exists, send `wallet:getAccounts` via `sendWalletMessage`
   4. If response is `wallet:accounts` -> set isLocked=false, accounts=resp.accounts, screen='main'
   5. If response is `wallet:error` -> set isLocked=true, screen='lock'
   6. Also load `network` from `chrome.storage.local.get('network')` (default 'mainnet')

   **Navigation rules:**
   - `push(screen)`: append to stack, update currentScreen
   - `pop()`: remove last from stack, currentScreen = last item (fallback 'welcome')
   - `replace(screen)`: replace last item in stack, update currentScreen
   - `reset(screen)`: clear stack, set to `[screen]`, update currentScreen (for post-onboarding transitions)
